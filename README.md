# PMBOK Summary Web

Este proyecto es una aplicación web que proporciona un resumen de la Guía PMBOK (Project Management Body of Knowledge). La aplicación está diseñada para ayudar a los profesionales de la gestión de proyectos a comprender los conceptos clave y las mejores prácticas descritas en la guía.

## Estructura del Proyecto

El proyecto contiene los siguientes archivos:

- **public/index.html**: La página principal de la aplicación, que incluye la estructura básica del HTML, el título y los enlaces a los estilos y scripts necesarios.
- **src/app.js**: Contiene la lógica principal de la aplicación, incluyendo funciones para manejar la interacción del usuario y cargar el resumen de la guía PMBOK.
- **src/styles.css**: Define los estilos CSS para la aplicación, incluyendo colores, fuentes y diseño visual.
- **package.json**: Archivo de configuración para npm que lista las dependencias y scripts necesarios para el proyecto.

## Instalación

1. Clona el repositorio en tu máquina local.
2. Navega al directorio del proyecto.
3. Ejecuta `npm install` para instalar las dependencias necesarias.

## Ejecución

Para ejecutar la aplicación, puedes usar un servidor local. Una opción es utilizar el paquete `http-server` de npm:

1. Instala `http-server` globalmente si no lo tienes: `npm install -g http-server`.
2. Desde el directorio del proyecto, ejecuta `http-server public`.
3. Abre tu navegador y visita `http://localhost:8080` para ver la aplicación en acción.
